# Sylhet Boat Club Limited (SBCL)

Welcome to the official repository of the **Sylhet Boat Club Limited** website. This is a premium, world-class frontend application built using **React** and **Vite**, featuring beautiful custom styling, ambient animations, and optimized media assets.

## Technology Stack

- **Core**: React 18, Vite 5
- **Styling**: Vanilla CSS, Flexbox, CSS Grid
- **Animations**: Framer Motion
- **Libraries**: Lucide React (Icons), Swiper (Sliders), React CountUp (Interactive Counters)

## Features

- **Hero Slider & Video**: High-end photorealistic 3D renders with a local video showcase modal playing the official SBCL film.
- **Real Client Assets**: Cropped and converted high-resolution renders from the master plan and logo assets.
- **Interactive Facilities Map**: Floor-by-floor breakdown of the architectural master plan.
- **Events & Notice Board**: Real event updates, custom tickers, and championships.
- **Clean Structure**: Standard code separation and component modularity.

## Getting Started

### Installation

```bash
npm install
```

### Run Locally

```bash
npm run dev
```

### Production Build

```bash
npm run build
```
